개요 예제 작성 팁 정리

개요

이 템플릿은 프로그래밍 포스트용 기본 구조입니다. 제목, 리드, 목차, 코드 블록 규칙만 유지하고 본문은 주제에 맞게 바꿔 쓰면 됩니다.

코드 블록 규칙

`<pre><code class="language-...">...</code></pre>` 형식만 지키면 Prism.js, line numbers, copy button, mac 스타일 헤더가 공통 자산으로 자동 적용됩니다.

예제

Java 예제:

List<String> names = List.of("Kim", "Lee", "Park");

for (String name : names) {
    System.out.println(name);
}

Bash 예제:

python3 -m http.server 8080

SQL 예제:

SELECT id, title, created_at
FROM posts
ORDER BY created_at DESC;

작성 팁

  • 코드 블록에는 가능한 한 `language-*` 클래스를 명시합니다.
  • 한 글 안에서 언어가 섞이면 블록마다 다른 언어 클래스를 붙입니다.
  • 인라인 코드는 `...`만 사용하면 됩니다.
  • 이미지를 쓰면 `assets/images/posts/programming/java/<slug>/...`처럼 언어 폴더 아래에 둡니다.
  • 새 글을 만들면 `posts.json` 메타도 같이 추가합니다.

정리

이 파일을 복사해서 새 프로그래밍 포스트를 만들고, 코드 블록에는 언어 클래스만 붙이면 공통 코드 스타일이 그대로 적용됩니다.

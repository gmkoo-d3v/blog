DDL, DML, DCL, TCL 한 번에 정리하기

SQL 명령어는 역할에 따라 DDL, DML, DCL, TCL로 나눠서 보면 외우기 편합니다. 자주 나오는 대표 명령어만 짧게 정리합니다.

2026-03-10 DB / SQL Basics

DDL

Data Definition Language

데이터베이스 객체의 구조를 정의, 변경, 삭제할 때 사용합니다.

CREATE
ALTER
DROP
TRUNCATE

DML

Data Manipulation Language

테이블 안의 데이터를 조회, 추가, 수정, 삭제할 때 사용합니다.

SELECT
INSERT
UPDATE
DELETE

DCL

Data Control Language

사용자 권한을 부여하거나 회수할 때 사용합니다.

GRANT
REVOKE

TCL

Transaction Control Language

트랜잭션을 확정하거나 되돌릴 때 사용합니다.

COMMIT
ROLLBACK
CHECKPOINT
메모

`CHECKPOINT`는 DBMS에 따라 관리 명령이나 트랜잭션 관련 명령으로 분류 방식이 조금 다를 수 있습니다.

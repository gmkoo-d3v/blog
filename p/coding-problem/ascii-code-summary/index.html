Coding Problem

정처기 아스키코드
필수 암기 정리

정처기 및 코딩 테스트에서 자주 등장하는 필수 아스키코드(ASCII) 값을 빠르게 복습할 수 있도록 표로 정리.

2026-03-21 CS Basics 정처기, 아스키코드, ASCII
개요 필수 아스키코드 표 대소문자 변환 공식 정리

개요

아스키코드(ASCII)는 미국 정보교환 표준 부호로, 알파벳, 숫자, 특수문자, 제어문자를 7비트(0~127)로 매핑한 문자 인코딩 방식이다.

핵심 문자 8개와 대소문자 간의 관계는 반드시 기억해 두어야 한다.

필수 아스키코드 표

문자 / 제어문자 의미 10진수 값
null (\0) 널 문자 (문자열의 끝) 0
tab (\t) 수평 탭 9
lf (\n) Line Feed (줄 바꿈) 10
cr (\r) Carriage Return (커서를 맨 앞으로) 13
space ( ) 공백 (띄어쓰기) 32
'0' 숫자 0 48
'A' 대문자 A 65
'a' 소문자 a 97

대소문자 변환 공식

알파벳의 아스키코드 값에서 가장 중요한 성질 중 하나는 소문자와 대문자의 값 차이가 항상 32라는 점이다.

핵심 공식

소문자 = 대문자 + 32

예를 들어, 대문자 'A'는 65이고, 소문자 'a'는 97이다. (65 + 32 = 97)

따라서 코딩을 할 때 대문자를 소문자로 바꾸려면 + 32를 하고, 소문자를 대문자로 바꾸려면 - 32를 하면 된다.

정리

  • 숫자 0은 48부터 시작한다.
  • 대문자 A는 65부터 시작한다.
  • 소문자 a는 97부터 시작한다.
  • 대문자와 소문자의 차이는 32이다. (소문자 = 대문자 + 32)
  • 자주 쓰는 제어 문자인 null은 0, tab은 9, lf(줄바꿈)는 10, cr은 13, space(공백)는 32이다.
